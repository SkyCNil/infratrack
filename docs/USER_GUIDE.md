# ITAM User Guide

## Login
1. Open application URL.
2. Enter username and password.
3. If prompted, change password on first login.

## Create Asset
1. Open `Create Asset`.
2. Fill required fields.
3. Submit to generate asset and QR label.

## Bulk Upload
1. Download sample CSV from `Create Asset`.
2. Fill rows using template columns.
3. Upload CSV/XLSX from Bulk Upload.

## Assign/Return/Repair
1. Open `Asset Assignment`.
2. Enter asset ID and assignee details.
3. Add mandatory remarks.
4. Use Assign, Return to Inventory, or Move to Repair.

## Search and Details
1. Open `Assets`.
2. Search by asset ID, model, serial, user.
3. Click asset ID to open full details and timeline.

## Reports
Open `Reports` and download CSV files required for business review.
