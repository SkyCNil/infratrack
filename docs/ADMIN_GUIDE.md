# ITAM Admin Guide

## Core Responsibilities
- Manage user IDs and roles.
- Configure masters (departments, locations, asset types, statuses).
- Review reports and audit logs.
- Handle lifecycle actions: lost, end-of-life, scrapped.

## Roles
- `Admin`: full access.
- `ITUser`: assign/return/repair, location updates.
- `Viewer`: read-only/report access.

## User Management
- Create users from Users menu.
- New users must change password on first login.
- Disable users when they leave organization.

## Asset Governance
- Asset ID auto-generated by nomenclature.
- Assignment and lifecycle actions require remarks.
- Timeline captures assignment and audit events.

## Reports
Use Reports menu for CSV exports:
- Asset Register
- Current Assignments
- Assignment History
- Lifecycle Events
- Assets by Status
- Assets by Department
- Warranty Expiry
